The aim of this project is to visualize academic and professional relationships of committee members of the EuroVis 2021 conference.

I would like to extend my sincerest thanks to Professor Elisabeta Marai for her guidance throughout the course of the project. I would also like to give a shoutout to Waleed Hasan for his help with data gathering without whose work, this project would not have been possible.

This project was built using several Open Source components. The source code of these open source components along with license information can be found below. I acknowledge and am grateful to these developers for their contributions to the open source community.
